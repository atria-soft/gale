What is GALE?

GALE, or "Graphic Abstraction Layer for Ewol" is a simple Interface with the OS Gui interface (openGL) and basic interface. For example, on android we use java interface, on linux we use X11, .... Gale abstract all of this and provide an inetrface in C++ on all of these platform. Gale abstact the Keybord, mouse and touch event on all of these platform (when availlable: no touch on linux X11)

Gale is a low level abstraction interface. you can user upper interface:

  • EWOL It provide some basic widgets and gui management

With Gale you have the Graphyc abstraction. We provide some abstraction for the audio interface:

  • audio-orchastra That is the dual of gale but for audio.
  • audio-river That is the dual of ewol but for audio.

Where can I use it?

Everywhere! GALE is cross-platform devolopped to support bases OS:

  • Linux (X11) (mouse)
  • Windows (mouse) (build on linux...)
  • MacOs (mouse)
  • Android (mouse + touch)
  • IOs (touch)

What languages are supported?

GALE is written in C++.
